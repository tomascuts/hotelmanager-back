# Hotel Manager - Sistema de Gestión de Accesos en Agencia Hotelera  

## [Introducción]  
Este proyecto tiene como objetivo desarrollar un sistema para la gestión eficiente de accesos a las distintas instalaciones de un hotel, implementando restricciones de uso y un sistema de cobros automatizado para mejorar la experiencia del cliente y optimizar los recursos del hotel.  

---

## [Objetivo]  
El sistema verifica los permisos de acceso de los huéspedes, asegurando que solo aquellos autorizados puedan ingresar a ciertas áreas del hotel. Además, automatiza el manejo de saldos y tareas administrativas, contribuyendo a:  

- **Reducir conflictos entre huéspedes.**  
- **Garantizar el cumplimiento de políticas de uso.**  

---

## [Características del Sistema]  

### [Instalaciones]  
- **Gratuitas:**  
  - Spa  
  - Comedor  
  - Casino  
- **Con costo adicional:**  
  - Bar  
  - Pelotero  

---

### [Restricciones]  
- **Casino y Bar:** Edad mínima requerida.  
- **Pelotero:** Edad máxima permitida.  
- **Comedor:** Capacidad máxima.  
- **Spa:** Límite de accesos por usuario.
